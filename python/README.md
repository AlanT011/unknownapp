# Course Enrollment System (Python Version)

This is a Python equivalent of the Java Course Enrollment System.

## How to Run

```bash
python main.py
```

## Features

- Student and Admin login
- Course catalog viewing
- Enrollment and dropping courses
- Schedule management
- Billing summaries
- Admin functions for managing students and courses

## Data Storage

Data is stored in JSON files in the `data/` directory:
- `data/students.json`
- `data/courses.json`
